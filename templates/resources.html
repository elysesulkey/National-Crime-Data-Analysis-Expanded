Crime Analysis Data Resources

USA Crime Data 2018-2019

Context

This table provides the number of offenses known to law enforcement for the first six months of 2018 and 2019 in cities with populations of 100,000 and over.

Content

The table contains 552 rows and 14 columns. Crime has been divided into following cases: 'Violent Crime', 'Murder', 'Rape', 'Robbery', 'Aggravated assault', 'Property Crime', 'Burglary', 'Larceny Theft', 'Motor Vehicle Theft', 'Arson'.

Data Source

All the credit for this data goes to the Federal Bureau of Investigation. For more information regarding the data refer here.
